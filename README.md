
# Knowledge AI - Research Sanctuary

Developed by **Oussama SEBROU**, this platform is an elite research terminal for axiomatic wisdom extraction from PDFs.

## Deployment Instructions

### 1. GitHub
- Create a new repository on GitHub.
- Push all files to the repository.

### 2. Render.com
- Create a **New Static Site**.
- Connect your GitHub repository.
- **Build Command:** `npm install && npm run build`
- **Publish Directory:** `dist`
- **Environment Variables:**
  - Add `API_KEY`: Your Google Gemini API Key.

## Features
- **3D Axiom Cards:** High-fidelity conceptual visualization.
- **Vast Chat Sanctuary:** Deep interrogation of manuscripts.
- **Framecode Syntax:** High-vibrancy coding interface.
- **Dual Language:** Full Support for Arabic (AR) and English (EN).
